Este repositório contém ferramentas de calculadora simples desenvolvidas durante o aprendizado de lógica de programação e automação em ambiente Linux.
1-Calculadora em Shell script feita pelo WSL

Funcionalidades: Soma, Subtração, Multiplicação e Divisão.
Destaques: Possui tratamento para erro de divisão por zero e formatação de casas decimais para divisões.
Como rodar: ```bash chmod +x calculadora.sh ./calculadora.sh

2-Calculadora em Python melhorada com while loop

Funcionalidades: Mesmas operações da versão Shell, com adição de um loop de repetição.
Destaques: * Uso de while True para realizar múltiplos cálculos sem fechar o programa.
Tratamento de exceções (try/except) para evitar que o programa trave se o usuário digitar letras.
Como rodar: ```bash python3 calculadora.py

Linguagens Utilizadas
Bash / Shell Script: Para automação e interação direta com o sistema.
Python 3: Para lógica de programação mais complexa e fluida.
Git & GitHub: Para controle de versão e hospedagem do código.
